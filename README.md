# Regresi Bayesian Menggunakan Rstan
Project ini melakukan regresi bayesian dengan menggunakan package **Rstan.**

Data yang digunakan adalah Data Kanker Serviks dan akan dilakukan regresi terhadap variabel Y. 
Akan dilakukan beberapa model regresi kemudian dibandingkan dan dipilih yang terbaik berdasarkan nilai DIC model. 

**Langkah penelitian:**
1. Eksplorasi Data
2. Uji Distribusi
3. Pemilhan Distribusi (untuk model regresi)
4. Permodelan Regresi
5. Pemilihan Model Terbaik.
